About

Hi! I'm Namkha

I'm a Robotics Engineer specialized in mobile robot control, embedded software and machine learning. I love to create intelligent systems that can optimize human productivity.

Over the years, I've had the pleasure of working in fast-paced startup environments, as well as established research teams developing robotic technology for Fortune 100 stakeholders, such as the IBOT personal mobility device and the FedEx Sameday autonomous delivery bot . Due to my slightly nonlinear educational trajectory, I am able to apply an interdisciplinary, broad systems-level approach towards my work.

My core competencies include:
* Developing linear and non-linear controllers for robotic applications
* Implementing Deep Learning (Tensorflow/Keras) and Reinforcement Learning for adaptive learning tasks
* Localization, mapping and path planning for autonomous mobile robots using the Robot Operating System (ROS)
* Programming software for embedded systems in C/C++
